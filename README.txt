How to run:

Main class: cz.vesely.game.Main
only client, server not done

default server connection is: 127.0.0.1:6930
can be changed in GameLogic class

To run outside of IDE do:
mvn clean package
copy target/Game-0.0.1-SNAPSHIT.jar, target/libs, shaders, textures, firecode.tff to a dedicated folder
run as: java -jar ./Game-0.0.1-SNAPSHIT.jar
